# Sistema_Locadora_de_Automoveis


Sistema web símples de uma locadora de autómóveis.


#Funcionalidades: 

-Cadastro e edição de usuários;

-Cadastro e edição de veículos;

-Alocação/desalocação de veículos;



#Tecnologias utilizadas:

-Node.js;

-Express.js;

-Sequelize ORM;

-MySQL;

-Bootstrap;

-Arquitetura MVC.
